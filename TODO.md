# AllKeyShop Price Monitor — Archived Roadmap

> **Status: ARCHIVED** — Extension v1.2.0 is the final release. The items below are ideas that were not implemented.

## Future Ideas (Not Implemented)

- Price history with mini-chart (sparkline) in game cards
- List sorting by name, price, update date, threshold
- Search/filter bar in the monitored games list
- Import from Playnite Library
- Visual price thresholds (green/red)
- Multi-currency support (USD, GBP, etc.)
- Steam wishlist sync
- Edition comparison (Standard, Deluxe, GOTY, etc.)
- CSV/JSON export
- Toast notification click-to-open
- List sharing via link
- Discord/Telegram webhook
- Plugin API for other extensions

## Technical Improvements (Not Implemented)

- Graceful handling of AllKeyShop HTML structure changes
- Retry with exponential backoff for HTTP requests
- Smarter cache invalidation
- DateTimeOffset for timezone handling
- Unit tests
